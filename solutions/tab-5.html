Solution: XSS in Legacy Code

The HTML is safe to serve to users because:

  • Our HTML templates have been reviewed so that the distinction
    between contributed content and site content is clear to users.
  • The html template tag only lets TrustedHtml values specify
    markup elements. Other values can only specify HTML text nodes.
  • Only modules on the TrustedHtmlProducer list can create
    TrustedHtml values    .
  • All such modules have been carefully reviewed.
